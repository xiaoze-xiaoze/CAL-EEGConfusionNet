# CAL-EEGConfusionNet

基于 CAL-EEG 数据集的学习混淆状态分类项目，使用深度学习识别学习者的混淆程度。

### 数据预处理

##### 基线校正: 对每个样本减去时间维度的均值，消除基线漂移

##### 标准化: 按通道计算标准差并归一化，使不同通道的信号幅度可比

### 模型架构 (EEGConfusionNet)

#### 多尺度卷积层
##### 使用 3×3、5×5、7×7 三种卷积核并行提取不同时间尺度的特征来捕获短期、中期、长期的脑电信号模式

#### GLU 门控单元
##### 通过门控机制自适应选择重要特征以此来抑制噪声，增强有效信号

#### 通道注意力机制
##### 学习不同脑电通道的重要性权重以此突出与混淆状态相关的关键通道

#### 双向 LSTM
##### 捕获时间序列的前后依赖关系建模脑电信号的时序动态变化

#### 时间注意力机制
##### 对不同时间步赋予不同权重以此来聚焦于混淆状态的关键时刻

#### GRN 门控残差网络
##### 通过残差连接和门控机制稳定训练防止梯度消失提升模型表达能力

#### 分类头
##### 全连接层输出三分类结果：Confused / Not Confused / Other

### 训练策略
##### 使用 Adam 优化器 + 学习率衰减

###### 交叉熵损失函数

###### 训练集/验证集划分（80% / 20%）

###### 最终在全部数据上重新训练并导出 ONNX 模型
